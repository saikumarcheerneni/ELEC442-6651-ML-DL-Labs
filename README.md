## Labs

### Lab 5 — SVM Classification
- Dataset: make_moons (1000 data points)
- Models: Linear SVM and RBF SVM
- Evaluation: 5-Fold Cross Validation
- Key concepts: Underfitting, Linear Separability, Model Capacity
- Results: RBF achieved 98% test accuracy vs 85.5% for Linear

### Lab 6 — MLP Neural Network (Coming Soon)
